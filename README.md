# Projet Blocus

Application web d'aide à l'étude pour étudiants.

## Stack Technique

- **Frontend**: HTML, CSS (Tailwind), JavaScript vanilla
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Déploiement**: Vercel

## Installation Locale

1. Clone le repo
2. Configure les variables d'environnement (`.env.example`)
3. Lance un serveur local : `npx vite`

## Déploiement

Le site est automatiquement déployé sur Vercel à chaque push sur la branche principale.

## Base de Données

Exécute les migrations SQL dans `supabase/` pour initialiser la base de données.
